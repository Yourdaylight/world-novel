# 设计系统 — WorldEngine

## 产品定位
- **产品描述:** 多Agent小说生成系统，通过独立AI角色的涌现行为创造具有心理真实感的长篇网文
- **目标用户:** 中国网文创作者、TRPG世界观爱好者、AI创作探索者
- **产品领域:** AI辅助创作 / 世界模拟引擎
- **项目类型:** Web应用仪表板 (Vue 3 + Element Plus + SCSS)

## 美学方向
- **方向:** 史官的仪器 (The Historian's Instrument)
- **装饰等级:** 有意识的 (intentional) — 数据上的微光效果，账簿式线条分隔，不使用装饰性色块或渐变。排版承担90%的视觉工作
- **氛围:** 一个见证过千个世界的造物主控制台。温暖而权威，像烛光照亮古卷，而非冷冰冰的SaaS后台。用户打开界面的感受应该是"我创造了某种永恒之物"
- **关键决策:**
  - 摒弃卡片布局，采用账簿式分区 — 内容以1px线条分隔，非浮起的矩形
  - 温暖琥珀色取代冷蓝色作为主要强调色 — "有什么正在发生"而非"这是信息"
  - 叙事内容使用衬线字体 — 章节文本和史官对话应有文学质感

## 排版

### 字体选择
- **标题/展示:** Cormorant Garamond (Light/Regular/Semibold) — 16世纪古典复兴字体，极端笔画对比。72px下的世界名称如同魔典扉页，诡异而权威
- **正文/叙事:** Source Serif 4 (Variable) — Adobe文学衬线体，完整CJK字符支持。史官对话和章节文本应有真实书籍的阅读感
- **界面/标签:** Instrument Sans (400-700) — 为仪器显示设计，小尺寸清晰可读。说"控制面板"而非"SaaS产品"
- **数据/表格:** Berkeley Mono / JetBrains Mono — 独立风格等宽字体，带Art Deco气质。大尺寸数值如宣告而非读数。必须支持 tabular-nums
- **代码:** JetBrains Mono
- **加载方式:** Google Fonts CDN
  ```html
  <link href="https://fonts.googleapis.com/css2?family=Cormorant+Garamond:ital,wght@0,300;0,400;0,500;0,600;0,700;1,300;1,400;1,500&family=Source+Serif+4:ital,opsz,wght@0,8..60,200..900;1,8..60,200..900&family=Instrument+Sans:ital,wght@0,400..700;1,400..700&family=JetBrains+Mono:wght@400;500;600&display=swap" rel="stylesheet">
  ```

| sm | 13px | 界面文本、表格内容 |

### 行高规则
- 数据区域: `line-height: 1.2` — 紧凑
- 界面文本: `line-height: 1.5` — 标准
- 英文叙事: `line-height: 1.85` — 宽松
- 中文叙事: `line-height: 2.0` — 适应中文字符的方块结构
- 段落间距 (叙事区): `1.2em - 1.5em`

## 色彩

### 色彩方针
- **方式:** 温暖表达 — 琥珀铜色为主信号色，语义色用于区分模块与状态
- **核心理念:** 用温暖的紫黑色替代冷蓝灰。每个AI工具都是蓝色的。琥珀色意味着"有生命正在活动"

### CSS变量

```css
:root {
  /* === 基础背景 === */
  --bg-void:        #09080c;   /* 近黑色，紫色底调 — 墨水，非石板 */
  --bg-surface:     #12101a;   /* 深茄紫 — 像在书页之中 */
  --bg-elevated:    #1c1826;   /* 抬升层 — 薄边框面板 */
  --bg-glass:       rgba(28, 24, 38, 0.82);  /* 浮层面板 */
  --bg-scrim:       rgba(9, 8, 12, 0.94);    /* 遮罩层 */

  /* === 文字 === */
  --text-primary:   #ede6d8;   /* 羊皮纸白 — 温暖，非临床白 */
  --text-secondary: #9e8fa3;   /* 暖灰紫 */
  --text-muted:     #58505f;   /* 极低对比度，用于真正次要的信息 */
  --text-inverse:   #09080c;   /* 用于深色文字（亮底按钮上） */

  /* === 强调色 === */
  --accent-ember:     #d4793a;   /* 琥珀铜 — 主信号色：心跳，非通知 */
  --accent-ember-dim: rgba(212, 121, 58, 0.15);
  --accent-ember-glow: rgba(212, 121, 58, 0.4);

  --accent-jade:      #4ec994;   /* 翡翠绿 — 成长/创造/积极状态 */
  --accent-cinnabar:  #e04545;   /* 朱砂红 — 命运/权力/警告 */
  --accent-aurora:    #a67fd4;   /* 极光紫 — 记忆/伏笔/过去 */
  --accent-phosphor:  #c6e847;   /* 磷光绿 — "存活"状态，当前活跃Agent */
  --accent-silver:    #c8bfd4;   /* 银紫 — 冷色高光用于对比 */

  /* === 结构线 === */
  --border-ghost:   rgba(237, 230, 216, 0.05);  /* 几乎不可见的分隔线 */
  --border-rule:    rgba(237, 230, 216, 0.10);  /* 账簿线、区域分隔 */
  --border-active:  rgba(212, 121, 58, 0.45);   /* 聚焦元素的琥珀边框 */

  /* === 光效/阴影 === */
  --glow-ember: 0 0 24px rgba(212, 121, 58, 0.25), 0 0 64px rgba(212, 121, 58, 0.08);
  --glow-jade:  0 0 20px rgba(78, 201, 148, 0.2);
  --shadow-deep: 0 8px 40px rgba(9, 8, 12, 0.8), 0 2px 8px rgba(9, 8, 12, 0.6);
}
```

### 模块色彩分配
| 模块 | 色彩 | 含义 |
|------|------|------|
| 世界/创造 | 翡翠 `--accent-jade` | 自然、成长 |
| 角色/灵魂 | 琥珀 `--accent-ember` | 意识、活跃 |
| 时间线/命运 | 朱砂 `--accent-cinnabar` | 命运、权力 |
| 伏笔/记忆 | 极光 `--accent-aurora` | 神秘、过去 |
| 当前活跃 | 磷光 `--accent-phosphor` | 实时、存活 |
| 系统/辅助 | 暖灰紫 `--text-secondary` | 工具、基础设施 |

### 深色/浅色模式
- **策略:** 深色模式为默认（沉浸感优先）。浅色模式反转基础色调：背景变为温暖亚麻色 `#f5f0e8`，强调色降低饱和度10-20%
- 浅色模式变量见预览页CSS

## 间距

- **基础单位:** 8px
- **密度:** 非对称密度 — 数据区紧凑，叙事区宽松
- **比例尺:**

| 名称 | 值 | 用途 |
|------|-----|------|
| 2xs | 2px | 最小间隙 |
| xs | 4px | 紧凑元素间距 |
| sm | 8px | 基础间距 |
| md | 16px | 标准内边距 |
| lg | 24px | 区块间距 |
| xl | 32px | 大区域间距 |
| 2xl | 48px | 区段间距 |
| 3xl | 64px | 页面级间距 |

### 密度规则
- **数据区域** (情感表格、时间线、关系图谱): 紧凑。`line-height: 1.2`, 单元格 `padding: 6px 0`
- **叙事区域** (史官对话、章节文本): 宽松。`line-height: 1.85-2.0`, 段落 `margin-bottom: 1.2em`
- **两种密度的对比让用户感受到数据与故事的本质差异** — 这正是WorldEngine的核心张力

## 布局

- **方式:** 非对称分栏 + 账簿式内容流
- **核心原则:** 左重布局像阅读，居中布局像使用软件。WorldEngine应该像阅读。
- **栅格:**
  - 侧边栏: 200-240px 固定宽度
  - 内容区: 弹性填充
  - 大屏 (>1440px): 内容区最大宽度 1100px
  - 中屏 (768-1440px): 全宽减侧边栏
  - 小屏 (<768px): 侧边栏收起为顶部导航
- **最大内容宽度:** 1100px (叙事阅读舒适度)
- **圆角:**
  - 默认: 0px — 账簿风格不使用圆角
  - 例外: 用户头像 `border-radius: 50%`, 特定图表容器可用 `4px`
  - **不使用** 大圆角 (12px+) 或 `rounded-full` 按钮

### 账簿式布局规则
- **不使用卡片** — 不使用浮起的矩形 (`box-shadow` + `background` + `border-radius`)
- **使用规则线** — 内容区域以 `1px solid var(--border-rule)` 水平线分隔
- **标题作为水印** — 区域标题可以以低透明度大尺寸出现在内容背后
- **例外:** 关系图谱面板和控制面板保留表面背景 — 但作为薄边框玻璃面板，不作为浮起的卡片。它们是"观察世界的窗口"，不是数据的盒子

## 动效

- **方式:** 双速度系统 — 只有"瞬间"和"缓慢"，没有中间态
- **缓动函数:**
  - 进入: ease-out
  - 退出: ease-in
  - 移动: ease-out-expo (cubic-bezier(0.16, 1, 0.3, 1))
- **时长:**
  - 瞬间 (0ms): 状态切换、数据更新、开关切换 — 命运早已注定，无需过渡
  - 缓慢 (600-900ms, ease-out-expo): 视图切换、图谱重绘、模拟启动 — 世界运转的重量感
- **不使用** 200ms微动画、弹性弹跳、悬浮缩放
- **特殊:** 琥珀呼吸脉搏 — 当前活跃数据 (模拟回合数、信任分数) 使用 3s ease-in-out 无限循环的透明度脉搏动画

## 反AI套路清单

以下模式**绝对不使用**:
- 紫色/紫罗兰渐变作为默认强调色
- 三列特性展示 + 彩色圆圈图标
- 所有内容居中 + 均匀间距
- 所有元素统一使用大圆角
- 渐变按钮作为主要CTA
- 通用英雄区 (hero section) 配库存照片风格
- "为X打造" / "专为Y设计" 等营销文案模式
