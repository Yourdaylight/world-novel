## 核心理念

```
传统 AI 小说工具           WorldEngine
─────────────────    vs    ─────────────────
输入提纲 → 输出文本          构建世界 → 运行世界 → 采集故事
一次性生成                    持续演化
单线叙事                      多视角叙事
AI 是作者                     AI 是世界的居民
用户是读者                    用户是造物主
```

> 一个世界，无数故事。
