## Agent 体系（当前 + 规划）

### 现有 Agent（V3 已实现）

| Agent | 模型 | 角色定位 | 核心能力 |
|-------|------|---------|---------|
| **Director** | GPT-4o | 故事总规划师 | 大纲、角色、关系图、卷册结构 |
| **Character** | GPT-4o-mini | 角色意识 (×N) | 自主行动、情感变化、关系演化 |
| **Writer** | GPT-4o | 史官/叙述者 | 将行动日志转化为文学叙事 |
| **Reviewer** | GPT-4o | 伏笔审计官 | 检查伏笔质量、标记悬空线索 |
| **God** | GPT-4o | 命运之手 | 章间决策、时间推进、世界事件 |

### 规划新增 Agent

| Agent | 角色定位 | 产品价值 |
|-------|---------|---------|
| **WorldCartographer** | 世界地图生成器 | 输入世界观 → 生成关系地图/势力版图 |
| **Historian** | 多视角史官 | 同一世界历史，不同视角叙述 |
| **DialogueMaster** | 对话专家 | 更自然的角色对话，口语化方言化 |
| **PacingAnalyzer** | 节奏分析师 | 张力曲线分析，避免平铺直叙 |
| **ThemeGuardian** | 主题守护者 | 确保每章回扣核心三问 |

## 数据模型核心实体

```
World (世界)
 ├── WorldView (世界观)
 │    ├── PowerSystem (力量体系)
 │    ├── Location[] (地点)
 │    ├── Faction[] (势力)
 │    └── HistoryEvent[] (历史事件)
 │
 ├── Timeline (时间线)
 │    ├── Era[] (时代)
 │    └── TimelineEvent[] (时间线事件)
 │
 ├── Character[] (角色)
 │    ├── Profile (档案 + Big Five 性格)
 │    ├── Goals[] (目标: 公开/秘密)
 │    ├── EmotionalState (6维情感向量)
 │    ├── EpisodicMemory[] (情景记忆)
 │    ├── SemanticMemory[] (语义记忆+向量)
 │    └── Relationship[] (关系网络)
 │
 ├── StoryOutline (故事大纲)
 │    ├── Volume[] (卷册)
 │    ├── ChapterOutline[] (章节大纲)
 │    └── SceneBeat[] (场景节拍)
 │
 ├── Foreshadow[] (伏笔)
 ├── PlotThread[] (剧情线)
 │
 └── Output
      ├── CharacterAction[] (角色行动日志)
      ├── Chapter[] (渲染后章节)
      └── GodDecision[] (命运决策记录)
```

```
       ▼

       ▼

│  ┌──────────┐   ┌─────▼─────┐            │
│  │ 伏笔审计   │◀──│ 命运决策   │ ◀─ 用户可  │
│  │ 质量检查   │   │ God Agent │   干预/观察 │
│  └──────────┘   └───────────┘            │
│                       │                   │
│              ┌────────┴────────┐          │

       ▼

```

## Agent 文件系统（已实现，需扩展）

```
workspace/
└── {novel_id}/
    ├── novel.db                    # SQLite 核心数据库
    ├── agents/
    │   ├── {character_id}/
    │   │   ├── agent.md            # 身份档案 (自动生成)
    │   │   ├── soul.md             # 灵魂核心 (用户可编辑 ★)
    │   │   └── memories/
    │   │       ├── era_1.md        # 按时代归档的记忆摘要
    │   │       └── era_2.md
    │   │
    │   └── god/
    │       ├── agent.md            # 命运之手的规则
    │       ├── timeline.md         # 时间线可视化
    │       └── decisions.md        # 决策历史
    │
    ├── world/
    │   ├── worldview.md            # 世界观文档 (可编辑)
    │   ├── map.svg                 # 世界地图 (未来)
    │   └── rules.md               # 世界规则 (可编辑)
    │
    └── output/
        ├── novel_主角A视角.md
        ├── novel_反派B视角.md
        └── short_story_某事件.md
```

## 技术栈

| 层级 | 技术 | 选择理由 |
|------|------|---------|
| 前端 | Vue 3 + TS + Element Plus | 已实现，暗色主题 |
| 可视化 | ECharts + vis-network | 情感图表 + 关系图谱 |
| 编辑器 | md-editor-v3 | soul.md 编辑 |
| 后端 | FastAPI + WebSocket | 异步 + 实时推送 |
| 编排 | LangGraph | 有向图状态机，支持条件分支和循环 |
| LLM | OpenAI GPT-4o / 4o-mini | 主力模型，支持替换 |
| 存储 | SQLite (per world) | 轻量、文件级隔离 |
| 向量 | FastEmbed (bge-small-zh) | 语义记忆检索 |
| 部署 | Docker / 本地 | 初期本地优先 |
