## 详细旅程

### Step 1: 创建世界 — "造物时刻"

用户进入"新世界"向导，回答三个终极命题 + 基础设定：

```
╔══════════════════════════════════════════════╗
║           🌍 创造你的世界                      ║
╠══════════════════════════════════════════════╣
║                                              ║
║  终极命题 ① — 这个世界"是什么"？               ║
║  ┌────────────────────────────────────────┐  ║
║  │ 描述你世界的本质。力量的来源是什么？       │  ║
║  │ 例：一个灵气复苏的现代都市，人类觉醒了     │  ║
║  │ 与星辰共鸣的能力，但每次使用都会失去       │  ║
║  │ 一段记忆...                              │  ║
║  └────────────────────────────────────────┘  ║
║                                              ║
║  终极命题 ② — 这个世界"从何而来"？             ║
║  ┌────────────────────────────────────────┐  ║
║  │ 世界的起源和历史。什么事件塑造了现在？      │  ║
║  │ 例：三千年前的"星陨之夜"，第一批觉醒者     │  ║
║  │ 建立了秩序，但被后来的帝国覆灭...          │  ║
║  └────────────────────────────────────────┘  ║
║                                              ║
║  终极命题 ③ — 这个世界"往何处去"？             ║
║  ┌────────────────────────────────────────┐  ║
║  │ 世界的命运走向。核心矛盾的终极指向？        │  ║
║  │ 例：星辰之力正在枯竭，人类必须选择         │  ║
║  │ 是牺牲记忆延续力量，还是回归凡人...        │  ║
║  └────────────────────────────────────────┘  ║
║                                              ║
║  基础设定                                     ║
║  类型: [玄幻 ▾]  基调: [史诗悲壮 ▾]           ║
║  主要角色数: [5 ▾]  预计章节: [50 ▾]          ║
║                                              ║
║         [ 🌟 开始创世 ]                       ║
╚══════════════════════════════════════════════╝
```

**系统响应**：
1. Director Agent 基于三个命题生成完整世界观
2. 自动创建力量体系、势力、地点、历史事件
3. 自动生成 N 个角色 + 关系图
4. 生成时间线和卷册结构
5. 为每个角色创建 `agent.md` + `soul.md`
6. WebSocket 实时推送进度

**耗时**：约 2-5 分钟（GPT-4o）

### Step 2: 调校世界 — "调校时刻"

创世完成后，用户进入世界仪表板，可以自由编辑：

#### 2a. 世界观调校
```
┌─────────────────────────────────────────┐
│ 🌍 世界观编辑器                          │
│                                         │
│ ┌─ 力量体系 ──────────────────────────┐ │
│ │ 名称: 星魂共鸣                       │ │
│ │ 等级: 感应→凝聚→共鸣→星主→... [编辑] │ │
│ │ 规则: 每次使用消耗记忆 [编辑]         │ │
│ │ 禁忌: 不可逆转他人记忆 [编辑]         │ │
│ └──────────────────────────────────────┘ │
│                                         │
│ ┌─ 势力版图 ──────────────────────────┐ │
│ │  [势力关系图 / 可拖拽节点]            │ │
│ │  帝国 ←对立→ 觉醒者联盟              │ │
│ │  黑市 ←依附→ 帝国                    │ │
│ └──────────────────────────────────────┘ │
└─────────────────────────────────────────┘
```

#### 2b. 角色灵魂编辑（核心交互 ★）
```
┌────────────────────────────────────────────────────┐
│ 👤 角色工坊 — 林墨 (主角)                            │
│                                                    │
│ ┌─ agent.md (只读) ──────┐ ┌─ soul.md (可编辑) ──┐ │
│ │ # 林墨                  │ │ # 灵魂核心           │ │
│ │ 性格: 内向、执着、...    │ │                     │ │
│ │ 背景: 孤儿，被师父收养  │ │ ## 价值观            │ │
│ │ 目标:                   │ │ - 记忆比力量更重要   │ │
│ │  - 公开: 找回失去的记忆 │ │ - 不会为了变强牺牲   │ │
│ │  - 秘密: 杀死背叛师父   │ │   他人的记忆         │ │
│ │    的人                 │ │                     │ │
│ │ 五维性格:               │ │ ## 底线              │ │
│ │  开放性: 0.7            │ │ - 绝不主动读取他人   │ │
│ │  尽责性: 0.8            │ │   记忆              │ │
│ │  外向性: 0.3            │ │ - 即使面对仇人       │ │
│ │  宜人性: 0.6            │ │                     │ │
│ │  神经质: 0.5            │ │ ## 内心冲突          │ │
│ │                        │ │ 越强大越孤独，越      │ │
│ │                        │ │ 接近真相越痛苦       │ │
│ │                        │ │                     │ │
│ │                        │ │ [💾 保存并同步]      │ │
│ └────────────────────────┘ └─────────────────────┘ │
│                                                    │

```

**关键设计**：用户通过编辑 `soul.md` 来影响角色的深层行为模式。
这不是"写剧本"，而是"塑造灵魂"——角色会基于这些价值观自主做出选择。

**关键交互**：
- **纯观看模式**：世界自动运行，用户只看结果
- **干预模式**：每章结束时可以审批/修改 God Agent 的决策
- **暂停/恢复**：任意时刻暂停，修改 soul.md 后恢复

---

### Step 5: 持续演化

世界不会"完结"。用户可以：

| 操作 | 说明 |
|------|------|
| **继续运行** | 追加 10/20/50 章，世界继续演化 |
| **投放新角色** | 在已运行的世界中加入新角色 |
| **创建分支** | 从某个检查点分叉出平行世界 |
| **修改规则** | 改变世界规则，观察蝴蝶效应 |
| **跨世界联动** | (远期) 不同世界的角色交叉 |

---

## 用户心智模型

```
用户不是在"写小说"
用户是在"经营一个世界"

不是 "我要写一个什么样的故事"
而是 "我要创造一个什么样的世界，看看会发生什么"

不是 "AI 帮我写"
而是 "AI 帮我运行，我来做造物主"
```
