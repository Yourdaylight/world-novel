# 08 — 里程碑路线图

## 版本演进

```
当前位置
   │
   ▼
V3 (已完成) ─── 时间线系统 + God Agent + Agent文件 + 前端Vue迁移
   │
   ▼
V4 (下一步) ─── 三个命题向导 + 世界观编辑 + 用户账户 + 工作空间隔离
   │
   ▼
V5 ──────────── 多视角采集 + 故事采集页 + 命运干预 + 导出增强
   │
   ▼
V6 ──────────── 世界分支 + 模板市场 + 团队协作 + 商业化上线
   │
   ▼
V7+ ─────────── 世界地图 + 高级Agent + 跨世界联动 + 移动端
```

## V4 — "造物主" (8-10 周)

> **主题**: 让用户真正成为世界的创造者

### Milestone 4.1 — 三个命题创世向导 (2 周)

| 任务 | 工作量 | 说明 |
|------|--------|------|
| 创世向导 UI (4步骤) | 前端 3d | Step1-3 三个命题 + Step4 参数确认 |
| AI 实时反馈接口 | 后端 2d | 用户输入 → GPT 实时分析返回建议 |
| 命题数据模型 | 后端 1d | 三个命题持久化到 DB 和世界观文件 |
| 命题注入 Director Prompt | 后端 1d | 让 Director 基于三个命题规划故事 |
| 首页/世界列表页 | 前端 2d | 世界卡片展示 + 创建入口 |
| 端到端测试 | 测试 1d | 从创建世界到生成完成 |

### Milestone 4.2 — 世界观在线编辑 (2 周)

| 任务 | 工作量 | 说明 |
|------|--------|------|
| 世界观编辑器组件 | 前端 3d | 力量体系/势力/地点/历史可编辑 |
| 世界观保存 API | 后端 1d | PUT /api/world 保存修改 |
| 世界规则 Markdown 编辑 | 前端 2d | 自由编辑世界额外规则 |
| 规则注入 Character Prompt | 后端 1d | 自定义规则传递给角色 Agent |
| 世界观变更历史 | 后端 1d | 记录修改日志 |
| 角色手动添加 | 前后端 2d | 在已有世界中投放新角色 |

### Milestone 4.3 — 用户账户 + 工作空间隔离 (3 周)

| 任务 | 工作量 | 说明 |
|------|--------|------|
| 用户表设计 | 后端 1d | SQLite 用户表 + JWT |
| 注册/登录 API | 后端 2d | 邮箱+密码, JWT 签发 |
| 登录/注册页面 | 前端 2d | UI + 表单验证 |
| 路由守卫 | 前端 1d | 未登录重定向到登录页 |
| 工作空间目录重构 | 后端 3d | data/{user_id}/worlds/{world_id}/ |
| API 所有权校验中间件 | 后端 2d | 所有 API 验证 user_id → world_id |
| Registry per-user 升级 | 后端 1d | 每个用户独立注册表 |
| 端到端测试 | 测试 2d | 多用户隔离验证 |

### Milestone 4.4 — 仪表板增强 (1 周)

| 任务 | 工作量 | 说明 |
|------|--------|------|
| 概览页增加三个命题展示 | 前端 1d | 显示世界的三个终极命题 |
| 概览页运行状态增强 | 前端 1d | God 决策数、世界事件数 |
| 角色页情感对比面板 | 前端 2d | 多角色情感同一图表对比 |
| 全局错误处理增强 | 前端 1d | 统一 API 错误提示 |

## V5 — "史官" (6-8 周)

> **主题**: 一个世界，无数故事

### Milestone 5.1 — 多视角故事采集引擎 (3 周)

| 任务 | 工作量 | 说明 |
|------|--------|------|
| Historian Agent 设计 | 后端 3d | 新 Agent: 基于角色视角重写场景 |
| 视角筛选逻辑 | 后端 2d | 筛选角色参与场景 + 信息屏蔽 |
| 情感着色叙述 | 后端 2d | 基于角色情感状态调整语气 |
| 采集 API | 后端 2d | POST /api/harvest 开始采集 |
| 采集进度 WebSocket | 后端 1d | 实时推送采集进度 |
| 故事采集页 UI | 前端 3d | 视角选择 + 设置 + 进度 + 阅读 |
| 已采集故事管理 | 前端 2d | 列表 + 阅读 + 导出 |

### Milestone 5.2 — 命运干预系统 (2 周)

| 任务 | 工作量 | 说明 |
|------|--------|------|
| God 决策审批 API | 后端 2d | GET 待审批决策 + POST 审批结果 |
| 用户干预指令注入 | 后端 2d | 用户额外指示传递给 God Agent |
| 干预审批弹窗 UI | 前端 2d | 决策展示 + 接受/拒绝/修改 |
| 运行模式切换 | 前后端 1d | 全自动/审批/通知三档 |
| 干预历史记录 | 前后端 1d | 记录所有用户干预 |

### Milestone 5.3 — 导出增强 (1 周)

| 任务 | 工作量 | 说明 |
|------|--------|------|
| EPUB 导出 | 后端 2d | 使用 ebooklib 生成 EPUB |
| PDF 导出 | 后端 2d | WeasyPrint 或 wkhtmltopdf |
| 导出格式选择 UI | 前端 1d | Markdown / EPUB / PDF |

### Milestone 5.4 — 短篇故事生成 (1 周)

| 任务 | 工作量 | 说明 |
|------|--------|------|
| 事件选择器 | 前端 2d | 从时间线选择事件 |
| 短篇展开 Agent | 后端 2d | 将单个事件展开为 1000-5000 字短篇 |
| 短篇管理 | 前端 1d | 列表 + 阅读 + 导出 |
