# 07 — 前端页面设计

## 视觉语言

### 设计基调
- **暗色系** — 沉浸感，"造物主的控制台"
- **Star Map 风格** — 背景微弱星辰粒子，暗示"世界"和"宇宙"
- **色彩体系** — 延续 slate/blue 调色板，功能色用于区分模块

### 色彩分配

| 模块 | 主色 | 含义 |
|------|------|------|
| 世界/World | 🟢 `#4ade80` 绿 | 自然、创造 |
| 角色/Character | 🔵 `#3b82f6` 蓝 | 意识、灵魂 |
| 时间线/Timeline | 🟡 `#fbbf24` 黄 | 命运、时间 |
| 伏笔/Foreshadow | 🟣 `#a78bfa` 紫 | 神秘、暗示 |
| 章节/Chapter | 🔷 `#22d3ee` 青 | 叙事、流动 |
| 命运/God | 🔴 `#ef4444` 红 | 命运、权力 |
| 控制/System | ⚪ `#94a3b8` 灰 | 系统、工具 |

## 响应式设计

| 断点 | 布局策略 |
|------|---------|
| ≥1440px | 完整布局，双栏面板 |
| 1024-1440px | 压缩侧边栏，单栏面板 |
| 768-1024px | Tab 改为侧边抽屉 |
| <768px | 暂不适配移动端（创作工具不适合小屏） |

## 关键交互原则

1. **即时反馈** — 世界运行中的每个事件都通过 WebSocket 实时推送
2. **渐进披露** — 初始只展示核心信息，点击/展开查看详情
3. **可逆操作** — 所有修改可撤销（检查点系统保障）
4. **造物主视角** — UI 语言始终暗示用户是"创造者"而非"使用者"
5. **暗色沉浸** — 全暗色主题，最小化视觉干扰，最大化沉浸感
