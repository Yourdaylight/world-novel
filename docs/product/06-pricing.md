# 06 — 商业模式与定价策略

## 定价哲学

```
用户付费的不是"文字"
用户付费的是"世界的运行时间"和"角色的思考次数"
```

**核心成本驱动因素**：
1. **LLM 调用次数** — 每个角色每个场景至少 2 次调用（行动+反应）
2. **LLM 模型等级** — GPT-4o vs GPT-4o-mini 成本差 10 倍
3. **世界复杂度** — 角色数 × 章节数 × 场景数 = 总调用量

## BYOK (Bring Your Own Key) 模式

针对有自己 API Key 的用户：

```
BYOK 订阅: ¥19/月
- 使用用户自己的 OpenAI/Anthropic API Key
- 平台只收取软件使用费
- 无 Token 额度限制
- 适合技术用户/开发者
```

这确保了：
- 降低平台的 LLM 成本风险
- 吸引对成本敏感的技术用户
- 用户对自己的 API 使用有完全控制权
